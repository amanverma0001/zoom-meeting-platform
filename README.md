# Zoom Clone - Premium Video Conferencing

A full-stack video conferencing web application built with React, Node.js, Socket.io, and WebRTC. Featuring a modern Glassmorphism design and persistent theme switching.

## 🚀 Features
- 🌑 **Sleek Dark Mode**: Global dark theme with high-contrast UI.
- ✨ **Glassmorphism**: Beautiful frosted-glass effects on Navbars and Chat.
- 🎥 **Premium Video Grid**: Responsive grid with participant name tags.
- 💬 **Modern Chat Interface**: Bubble-style chat with real-time updates.
- 🌗 **Theme Switcher**: Floating toggle to switch between light and dark modes.

## 🛠️ Setup & Installation

### Backend
1. Navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example`:
   ```bash
   cp .env.example .env
   ```
4. Add your **MongoDB URL** in the `.env` file.
5. Start the server:
   ```bash
   npm start
   ```

### Frontend
1. Navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## 🔐 Security
This project uses `dotenv` for environment variable management. Ensure your `.env` file is never pushed to public repositories.
